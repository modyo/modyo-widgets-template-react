# modyo-widgets-template-react

## Project settings

- npm
- Node 18
- Vite 4
- React 18
- Bootstrap 5

## Scripts

### Install dependencies

```
npm i
```

### Local development

```
npm run storybook
```

### Build project

```
npm run build
```

### Run tests

```
npm run test
```

### Run tests coverage

```
npm run test:coverage
```

### Run lint

```
npm run lint
```

### Run lint style

```
npm run lint:style
```

### Run fix lint style

```
npm run lint:style:fix
```
