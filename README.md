# modyo-widgets-template-react

## Project settings

- npm
- Node 18
- Vite 4
- React 18
- Bootstrap 5

## Scripts

### Install dependencies

```
npm i
```

### Local development

```
npm run serve
```

### Build project

```
npm run build
```

### Preview

```
npm run preview
```

### Push to Modyo (.env file required)

```
npm run modyo-serve
```
